# Eigenface

## Asennus ja käynnistys

Asenna riippuvuudet komennolla

```bash
poetry install
```

Mene hakemistoon src ja käynnistä ohjelma komennolla

```bash
poetry run python3 index.py
```

Suorita testit komennolla

```bash
poetry run pytest
```
